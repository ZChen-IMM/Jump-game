# Jump-game
Arduino/CPX compatible sound-controlled game.

To play:
1. Download all the files
2. Download the compatible .UF2 file from circuitpython
3. Install circuitpython (instructions in slides)
4. Drag code.py and drop into circuitpython drive
5. Install node in terminal
6. In one terminal tab: node server.js
7. In another terminal tab: http-server
8. Open the destination html and play. Create any sound to control the movement
